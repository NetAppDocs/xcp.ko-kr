== iSync 를 선택합니다

xCP NFS입니다 `isync` 명령은 소스와 대상을 비교하고 카탈로그 인덱스를 사용하지 않고 타겟의 차이점을 동기화합니다.

.구문
[source, cli]
----
xcp isync <source_ip_address>:/src <destination_ip_address>:/dest
----
